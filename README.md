# Dictionary 
It's  a fast, user-friendly, and dynamic dictionary interface that integrates seamlessly with Wordly’s language tools.

You can:
-Search for a word
-Retrieve information such as pronunciation, definitions, synonyms, and source details.

## Installation

### Clone repository
```bash 
git clone <your-username> Single-Page-Application
cd Single-Page-Application
```

## Install Rquirements
```bash
npm install
```

### Run Live server
-Will appear at port 5000
